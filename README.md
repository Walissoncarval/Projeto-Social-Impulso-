# Projeto Impulso Social

O Projeto Impulso Social é uma plataforma web dedicada a fortalecer o trabalho de ONGs, proporcionando visibilidade institucional, gestão de voluntariado, captação de recursos e transparência para doadores. Com estrutura responsiva, acessível e otimizada, conecta pessoas e causas para transformar vidas.

## Estrutura
- index.html — Página inicial
- projetos.html — Página de projetos sociais
- cadastro.html — Formulário de cadastro com validação HTML5
- css/style.css — Estilos
- js/scripts.js — Máscaras de input
- images/ — Imagens e placeholders

> Observação: as imagens incluídas são placeholders geradas automaticamente. Recomenda-se substituí-las por imagens royalty-free (Unsplash, Pexels, Pixabay) ou fotos de uso autorizado pela ONG.
